AURANT ACEX is an advanced, multimodal quantitative trading architecture designed for high-frequency and multi-horizon financial forecasting. The system integrates Temporal Fusion Transformers (TFT) with a time-series Mixture-of-Experts (MoE) framework to dynamically specialize across regimes, enabling adaptive modeling of non-stationary market behavior and structural breaks.

The pipeline ingests heterogeneous data streams, including OHLCV price data, derived technical indicators, macroeconomic signals, alternative datasets, and event-driven features. These inputs are encoded into unified multimodal embeddings via learned projection layers, enabling cross-domain feature interaction and temporal alignment. Data is stored and streamed using highly optimized compressed Parquet formats, ensuring scalable, low-latency access for large-scale backtesting and live inference.

At the core, the TFT backbone captures long- and short-term dependencies through attention mechanisms, gating layers, and interpretable variable selection networks. This is augmented by a hierarchical MoE layer, where specialized expert subnetworks focus on distinct temporal regimes (e.g., volatility clusters, momentum phases, mean-reversion periods). A learned gating network dynamically routes inputs to the most relevant experts, improving generalization and robustness.

The architecture incorporates multi-timescale temporal encoding, allowing simultaneous modeling of intraday, daily, and long-horizon trends. Cross-attention modules fuse multimodal embeddings with time-series representations, enabling context-aware predictions. Regularization strategies, including sparsity constraints and entropy penalties on expert selection, ensure efficient utilization of model capacity.

Training is performed using a hybrid objective combining predictive loss (e.g., directional accuracy, return regression) with risk-aware optimization metrics such as Sharpe ratio maximization and drawdown minimization. The system supports online learning and continual adaptation through rolling window retraining and reinforcement-style feedback loops.

AURANT ACEX is designed for both research and production deployment, with a modular architecture that supports distributed training, real-time inference, and seamless integration with execution systems. Its combination of transformer-based temporal modeling, expert specialization, and multimodal fusion enables state-of-the-art performance in complex financial environments characterized by noise, regime shifts, and high dimensionality.
